## What it is

A sight-words flashcard app I built because every other one I tried was either stuffed with ads or trying to harvest data from a five-year-old. This one isn't. Customize the word list, toggle random order, set auto-advance, enable text-to-speech, and go. All settings live in localStorage. There is no backend.

## Features

- Clean single-card interface with large, readable type
- Configurable word lists (save your own)
- Random order or sequential
- Auto-advance on a timer
- Text-to-speech via the Web Speech API with rate, pitch, and voice controls
- No accounts, no ads, no telemetry

## Tech

React 18 + Vite, shadcn/ui components on Radix primitives, Tailwind, and Capacitor so it can ship as an Android app alongside the web build.
